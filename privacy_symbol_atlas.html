Privacy Policy

《夢境象徵地圖 Symbol Atlas》隱私權政策

不要求帳號、不上傳夢境內容、不追蹤使用者。

最後更新：2026-04-10

感謝你使用《夢境象徵地圖 Symbol Atlas》。我們重視你的隱私，也盡量讓 App 保持單純：不要求登入、不放廣告、不追蹤使用者。

我們是否收集個人資料

目前版本的《夢境象徵地圖 Symbol Atlas》不會主動收集、上傳或分享你的個人資料。

App 不包含：

  • 帳號登入系統
  • 第三方廣告 SDK
  • 第三方分析 SDK
  • 使用者追蹤功能
  • 雲端同步功能
  • 位置、相機、麥克風或聯絡人存取

儲存在裝置上的資料

為了讓 App 正常運作，App 會在你的裝置本機儲存以下資料：

  • 你輸入或保存的夢境內容
  • 夢境象徵、標籤、分類與整理紀錄
  • 收藏、搜尋或瀏覽相關的本機紀錄
  • 設定偏好，例如語言、顯示與使用體驗相關設定

這些資料儲存在你的裝置上，不會由 App 自動傳送給我們或第三方。

夢境內容與本機保存

使用者輸入的夢境資料只儲存在裝置本機。《夢境象徵地圖 Symbol Atlas》不需要帳號，也不會將你的夢境內容上傳到開發者伺服器。

請依照自己的需求管理裝置安全、備份與檔案分享對象。

匯出與匯入功能

若你使用匯出或匯入功能，App 會使用本機 JSON 檔案處理資料。這些檔案由你主動儲存、選擇或分享。

如果你將 JSON 檔案分享給其他 App、服務或對象，後續資料處理方式會依照該 App、服務或接收方的隱私政策與使用方式而定。

客服與主動聯絡

如果你透過電子郵件或其他方式主動聯絡開發者，你提供的聯絡資訊與訊息內容只會用於回覆你的問題、處理回報或提供支援。

請不要在客服訊息中提供不必要的敏感個人資料或完整夢境內容。

兒童隱私

《夢境象徵地圖 Symbol Atlas》不會刻意收集兒童的個人資料。若你認為 App 意外收到兒童個人資料，請聯絡我們，我們會協助處理。

資料刪除

你可以在 App 內刪除夢境紀錄、清除本機資料，或透過刪除 App 的方式移除儲存在裝置上的 App 資料。

若你曾自行匯出 JSON 檔案，請記得同步管理或刪除那些由你儲存於裝置、雲端硬碟或其他位置的檔案。

第三方服務

目前版本沒有整合第三方分析、廣告、雲端資料庫或追蹤服務。

若未來版本加入需要傳輸資料的新功能，我們會更新本隱私權政策，並依照 App Store 規範調整資料揭露。

政策更新

我們可能會因功能調整、法規要求或送審需求而更新本政策。更新後的版本會標示新的「最後更新」日期。
